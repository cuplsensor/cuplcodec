Defines

TEMPRH¶

Last character of the URL version string if the URL contains both temperature and relative humidity measurands.

TEMPONLY¶

Last character of the URL version string if the URL contains only temperature measurands.

ENDSTOP_BYTE¶

Last character of the endstop. Must be URL safe according to RFC 1738.

void sample_init(unsigned int stat, bool err, bool temponly)¶

Initialise the sample state machine.

Parameters
  • stat: 16-bit status value.
  • err: Sets an error condition where data will not be logged to the URL circular buffer.
  • temponly: When true only temperature measurands will be recorded to the circular buffer on each call of sample_push().

static void loadboth(sdchars_t *sample, int meas1, int meas2)¶

Write both measurands of a sample.

Parameters

  • meas1: Measurand 1. Only the 12 least sigificant bits will be used.
  • meas2: Measurand 2. Only the 12 least sigificant bits will be used.

int sample_push(int meas1, int meas2)¶

Append a sample containing up to two measurands onto the circular buffer.

Return
1 if the cursor has moved from the end to the start and data are being overwritten. Otherwise 0.
Parameters
  • meas1: Measurand 1 e.g. temperature.
  • meas2: Measurand 2 e.g. relative humidity.
