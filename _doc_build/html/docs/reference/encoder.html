static int inbounds(int byteindex)¶

Checks if a byte index is within the bounds the buffer array.

This can be used to prevent the program from accessing memory that is out of bounds.

Return
0 if the index is less than the size of the buffer and can be accessed safely. Otherwise 1.
Parameters
  • byteindex: Index of a buffer byte that is to be read or written.

int ndef_writeblankurl(int qlenblks, char *statusb64, int *qstartblk, char ver)¶

Create a URL with a blank circular buffer.

The circular buffer is zero-padded with a base64 octet that decodes to 0,0,0,0,0,0.

Return
1 if qlenblks is not even.
Parameters
  • qlenblks: Length of the circular buffer URL parameter (q) in blocks.
  • statusb64: Pointer to the status field encoded in base 64.
  • qstartblk: Pointer to an integer into which this function writes the start block of the q URL parameter.
  • ver: Version identifier.
