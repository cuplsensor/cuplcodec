int ndef_writeblankurl ( int  qlenblks,
char *  statusb64,
int *  qstartblk 
)

Create a URL with a blank circular buffer.

Parameters
qlenblksLength of the circular buffer URL parameter (q) in blocks.
statusb64Pointer to the status field encoded in base 64.
qstartblkPointer to an integer into which this function writes the start block of the q URL parameter.
Returns
1 if qlenblks is not even.

The circular buffer is zero-padded with a base64 octet that decodes to 0,0,0,0,0,0.
